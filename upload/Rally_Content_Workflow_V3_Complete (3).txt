A.2 GATE MULTIPLIER FORMULA
--------------------------------------------------------------------------------

┌─────────────────────────────────────────────────────────────────────────────┐
│ GATE MULTIPLIER CALCULATION                                                 │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│ ═══════════════════════════════════════════════════════════════════════════ │
│                         STEP 1: GATE PASS & MULTIPLIER                      │
│ ═══════════════════════════════════════════════════════════════════════════ │
│                                                                             │
│ gate_pass = min(G₁, G₂, G₃, G₄) > 0                                         │
│                                                                             │
│ g_star = average(G₁, G₂, G₃, G₄)                                            │
│                                                                             │
│ M_gate = 1 + β × (g_star - 1)                                               │
│                                                                             │
│ Where:                                                                      │
│ • G₁-G₄ = Four gate scores (0-2 each)                                       │
│ • β = 0.5 (constant)                                                        │
│ • g_star = Average of 4 gate scores                                         │
│                                                                             │
│ ═══════════════════════════════════════════════════════════════════════════ │
│                         GATE SCORE IMPACT TABLE                             │
│ ═══════════════════════════════════════════════════════════════════════════ │
│                                                                             │

A.3 CAMPAIGN POINTS FORMULA
--------------------------------------------------------------------------------

┌─────────────────────────────────────────────────────────────────────────────┐
│ CAMPAIGN POINTS CALCULATION                                                 │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│ ═══════════════════════════════════════════════════════════════════════════ │
│                         STEP 2: CAMPAIGN POINTS (Q SCORE)                   │
│ ═══════════════════════════════════════════════════════════════════════════ │
│                                                                             │
│ Campaign_Points = M_gate × Σ(W[i] × normalized_metrics[i])                  │
│                                                                             │
│ Where:                                                                      │
│ • W = Vector of metric weights (0-1 for each of 11 metrics)                 │
│ • W[i] = 0 turns that metric OFF completely                                 │
│ • Higher W[i] = More impact from that metric                                │
│                                                                             │
│ NORMALIZED METRICS:                                                         │
│ ───────────────────                                                         │
│ • EP (Engagement Potential): 0-5 → normalized                               │
│ • TQ (Technical Quality): 0-5 → normalized                                  │
│ • RT (Retweets): log(RT+1)                                                  │
│ • LK (Likes): log(LK+1)                                                     │
│ • RP (Replies): log(RP+1)                                                   │
│ • QR (Quality of Replies): 0-1 (AI evaluated)                               │
│ • FR (Followers of Repliers): log(total_followers+1)                        │
│                                                                             │
│ ═══════════════════════════════════════════════════════════════════════════ │
│                         LOG SCALING EXPLAINED                               │
│ ═══════════════════════════════════════════════════════════════════════════ │
│                                                                             │
│ WHY LOG SCALING?                                                            │
│ ──────────────────                                                          │
│ • Prevents runaway scores from viral content                               │
│ • 100 RTs ≠ 10x score of 10 RTs                                            │
│ • Diminishing returns at high engagement                                   │
│                                                                             │
│ EXAMPLE:                                                                    │
│ ─────────                                                                   │
│ 10 RTs  → log(11)  = 1.04                                                   │
│ 100 RTs → log(101) = 2.00  (only ~2x higher, not 10x)                       │
│ 1000 RTs → log(1001) = 3.00 (only ~3x higher than 10 RTs)                   │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
